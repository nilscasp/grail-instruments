# Placeholder Images

This folder should contain the following images:

## Root Level
- `og-image.jpg` - 1200x630px - Open Graph image for social sharing
- `logo.png` - Logo file
- `favicon.ico` - Favicon
- `apple-touch-icon.png` - Apple touch icon

## /images/instruments/
Instrument product photos. Recommended:
- 1200x1200px square format
- High quality, neutral background
- Multiple angles per instrument

Example filenames:
- `ayasa-kurd-1.jpg`
- `ayasa-kurd-2.jpg`
- `yishama-celtic-1.jpg`

## /images/makers/
Maker profile images. Recommended:
- 400x400px or larger
- Can be logo or portrait

Example filenames:
- `ayasa.jpg`
- `yishama.jpg`
- `pantheon-steel.jpg`

## /images/journal/
Blog post cover images. Recommended:
- 1600x900px (16:9 aspect ratio)
- Atmospheric, evocative

Example filenames:
- `choosing-handpan.jpg`
- `listening-craft.jpg`
- `portal-explained.jpg`

## /images/
General images:
- `about-founder.jpg` - About page portrait
- `placeholder.jpg` - Default fallback

## /audio/
Sound recordings in MP3 format:
- `ayasa-kurd-sample.mp3`
- `yishama-celtic-sample.mp3`
- etc.
